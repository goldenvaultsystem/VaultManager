      金库管家软件使用银行U盾技术保护你的重要数据。金库管家简化了用户使用Windows原生bitlocker加密VHD文件的流程,增加银行U盾安全技术,帮助用户有效杜绝数据被暴力破解窃取风险,极大保护了用户的数据安全。
      金库管家生成的金库VHD文件也是Windows原生支持,可以在不使用金库管家情况下,直接在任何Windows环境中打开。
      金库管家主要应用于以下三个应用场景:
      1.提供可用于本地自用的大容量金库文件,使用金库文件时需搭配金钥硬件一起使用,用户可以新建本地自用大容量安全存储容器文件;
      2.提供可用于云端存储的中容量金库文件,使用金库文件时需搭配金钥硬件一起使用,用户可以在互联网安全存放重要数据无需担心泄密;

1. Product Purpose
      The Vault Manager software uses bank security key technology to protect your important data. It simplifies the process of using Windows native BitLocker to encrypt VHD files, adds bank-grade security key technology, helps users effectively eliminate the risk of data being stolen through brute-force attacks, and greatly enhances the security of user data.
      The vault VHD file generated by Vault Manager is natively supported by Windows and can be opened directly in any Windows environment without using Vault Manager.
      Vault Manager is primarily used in the following three application scenarios:
      1. Provides large-capacity hardware-encrypted vault files for local personal use. Security key hardware is required to access vault data. Users can create local personal-use hardware-encrypted secure storage container files;
      2. Provides medium-capacity public key encrypted vault files for cloud storage. Security key hardware is required to access vault data. Users can securely store important data on the internet without worrying about leaks;
      3. Provides small-capacity public key encrypted vault files for remote transmission, ensuring the confidentiality and integrity of important data during transmission. Users can securely send important data to their remote endpoints over the internet.
